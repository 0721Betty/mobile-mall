#### 移动端的真机调试
- 保证手机与电脑处于同一个WIFI环境中，也就是说手机可以访问到电脑的IP
- 查看电脑的无线网ip地址`ipconfig`
- 在项目的package.json文件中，在dev脚本中，添加一个--host指令，在这个指令后面加上上面查到的ip地址
- 在电脑上重新运行npm run dev
- 打开手机浏览器输入项目运行的地址
